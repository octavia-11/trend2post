# Trend2Post

Trend2Post 是一个面向本地小店的内容获客方案展示页，核心思路是从小红书、抖音、Instagram 等平台识别“小账号真爆款”，再拆解成小店当天就能发布的内容脚本和拍摄清单。

## 在线预览

[点击打开网页](https://htmlpreview.github.io/?https://github.com/octavia-11/trend2post/blob/main/index.html)

## 项目文件

- `trend2post_merged.html`：完整网页文件，可直接在浏览器中打开预览。
- `index.html`：网页入口文件，用于在线访问。

## 核心功能

- 展示 Trend2Post 的机会判断、用户证据、产品原型、商业模式与验证计划
- 模拟“小店 AI 管家”的内容生成流程
- 将爆款内容拆解为标题、画面、卖点、拍摄任务和发布建议
- 支持作为课程项目、创业计划或产品 Demo 页面展示

## 本地预览

直接双击打开：

```text
trend2post_merged.html
```

或在浏览器地址栏打开本地文件路径。
